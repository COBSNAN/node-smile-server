Getting Started

Download the SMILE Junction Server Admin apk to a teacher tablet, and then download the MySMILE apk to one ore more student tablets. The SMILE Junction Server is already running (if you can see this page). You can always check on server status to see what version is running.

Development

NIDE

NIDE is a web-based IDE for Node.js, designed with simplicity and ease-of-use in mind. NIDE can be used on the SMILEPlug for Tablet or Web-based develpment of services to run on the PlugComputer.
